# Сторонние компоненты — «Поток»

Приложение «Поток» использует перечисленное ниже стороннее ПО и модели.
Авторские права и лицензии сохраняются за их правообладателями.

## Модели

- **GigaAM-v3** (распознавание речи) — © ai-sage / SaluteDevelopers. Лицензия **MIT**.
  Используется ONNX-сборка `istupakov/gigaam-v3-onnx`.
  https://huggingface.co/ai-sage/GigaAM-v3
- **Silero VAD** (детекция речи) — © Silero Team. Лицензия **MIT**.
  https://github.com/snakers4/silero-vad

## Библиотеки

| Компонент | Лицензия | Назначение |
|---|---|---|
| PySide6 (Qt for Python) | **LGPL v3** | плавающий оверлей записи |
| onnxruntime | MIT | запуск ONNX-моделей |
| onnx-asr | MIT | обвязка ASR над GigaAM |
| numpy | BSD-3-Clause | обработка аудио |
| sounddevice | MIT | захват микрофона |
| pynput | LGPL v3 | глобальный хоткей и эмуляция ввода |
| pystray | LGPL v3 | иконка в системном трее |
| Pillow | MIT-CMU (HPND) | генерация иконок |
| pyperclip | BSD-3-Clause | буфер обмена |

## Замечание по LGPL

Компоненты под LGPL (PySide6, pynput, pystray) подключаются как динамические
библиотеки и могут быть заменены пользователем на совместимые версии.
Их исходный код доступен у соответствующих правообладателей по ссылкам выше /
в их официальных репозиториях.
