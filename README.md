## Что это

«Поток» — push-to-talk диктовщик на русском (аналог Whisper Flow) на модели **GigaAM-v3**. Зажал хоткей → продиктовал → текст появляется там, где курсор. Офлайн и приватно: распознавание на твоём компьютере, аудио живёт только в оперативной памяти, на диск ничего не пишется. Текст из коробки чистый — пунктуация, числа, вырезание заминок, словарь IT-терминов.

## Установка

Скачай из [Releases](../../releases):

- **Potok-Setup.exe** — установщик (рекомендуется): ярлык, автозапуск, удаление через «Программы и компоненты».
- **Potok-windows.zip** — portable: распакуй и запусти `Potok.exe`.

Python и зависимости не нужны — модель вшита, интернет тоже. Хоткей по умолчанию — **Ctrl+Space**.

> Windows может показать SmartScreen (приложение без подписи) — «Подробнее» → «Выполнить в любом случае».

## Как пользоваться

- **Запись:** зажми хоткей, говори, отпусти — текст вставится туда, где курсор.
- **Меню в трее:** режим hold/toggle, свой хоткей, автозапуск, выход.
